HMTL Cheat Sheet

1. a - Hyperlinks

Sirven para hacer un link de una página a otro recurso en la web, sea otro archivo en el servidor web, u otra página web.

Por ejemplo: Enlace a mercado libre

2. p

Es un elemento semantico que sirve para identificar párrafos.

Por ejemplo: Esto es un párrafo

3. h1, h2, h3, h4, h5, h6

Los headings son son elementos que se usan para dar jerarquías dentro del documento. No deben ser utilizados para fines estéticos (hacer más grande el texto). La mayor prioridad es de h1 al h6.

Por ejemplo:

Este es un heading h1

Este es un heading h2

Este es un heading h3

Este es un heading h4

Este es un heading h5
Este es un heading h6

4. div

El div se usa para agrupar contenido. No existe un valor semántico para la etiqueta div y se ocupa más comunmente para dar estilo con css.

Por ejemplo:

Esto es un div

5. span

El span se usa para dar estructura inline o dentro de linea

Por ejemplo:

esto es un span

6. Strong

Para utilizar negritas y al mismo el valor semántico de dar más importancia

Por ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. At doloremque expedita voluptas. Animi atque eveniet soluta dignissimos recusandae et ullam alias qui quam! Eum quo illo eius, error debitis voluptatum?

7. i

Se usa para escribir un texto en itálicas.

Por ejemplo:

Esto es un texto en itálicas

8. u

Se usa para mostrar el texto subrayado.

Por ejemplo:

Esto es un texto subrayado

9. b

Similar al strong se usa para escribir en negritas pero con menor énfasis, semánticamente hablando.

Por ejemplo:

Texto en negritas

10. Strike

Se usa para mostrar el texto tachado con una linea horizontal.

Por ejemplo:

Texto tachado

11. Small

El elemento small se usa para reducir el tamaño del texto.

Por ejemplo:

Tamaño reducido del texto

12. em

Similar a lo que sucede con el elemento i. Este elemento hace el texto en itálicas pero con un mayor énfasis a comparación del elemento i.

Por ejemplo:

Texto demostrativo

13. Header

El elemento header ayuda para dar estructura a la página web. Se usa para identificar el principio de una página. Puede incluir elementos de navegación.

Por ejemplo:

Dentro de este header hay un párrafo

14. Section

El elemento section es un elemento de seccionamiento genérico, diseñado para contener una parte de un documento temáticamente definida.

Por ejemplo:

Dentro de esta etiqueta section hay un párrafo

15. Article

El elemento de article se usa para identificar una sección con contenido independiente. De igual manera que el elemento section, article se usa para dar estructura al documento.

Por ejemplo:

Dentro de la etiqueta article hay un párrafo

16. Aside

Aside se usa para representar una sección que consiste en información que está indirectamente relacionada con el contenido de mayor importancia en el documento.

Por ejemplo:

Dentro de la etiqueta aside hay un párrafo. El contenido de este párrafo puede estar relacionado indirectamente con la información general del documento.

17. Footer

De igual manera que los 4 elementos anteriores, el elemento footer se utiliza para dar estructura al documento. En específico se utiliza para identificar el final de la página.

Por ejemplo:

Copyright © 1990-2014 Santiago Benitez, todos los derechos reservados.

18. nav

El elemento nav se usa para distinguir la sección del documento en donde se encuentran los links de navegación de la página.

Por ejemplo:

  • home
  • about
  • contact

19. img

Se utiliza para insertar imágenes

Por ejemplo:

20. br

El elemento br produce un salto de linea

Por ejemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Rem, perferendis veniam? (salto de linea)
Alias reiciendis earum corrupti minima voluptatem facere repellat soluta ipsum impedit veritatis aliquam iure perferendis, magnam vitae nihil numquam.

21. hr

Este elemento representa un cambio de tema entre párrafos

Por ejemplo:

Se ve representado con una linea

22. form

Se utiliza para recibir información de usuarios.

Por ejemplo:

23. table

El elemento table nos ayuda a crear tablas, definiendo el encabezado, las filas y las columnas

Por ejemplo:

Mes Ahorros
Enero $100
Febrero $80

24. ul

Este elemento nos ayuda a definir listas con elementos no ordenados.

Por ejemplo:

  • uno
  • dos
  • tres

25. ol

Este elemento nos ayuda a definir listas con elementos ordenados.

Por ejemplo:

  1. uno
  2. dos
  3. tres
