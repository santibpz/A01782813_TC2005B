Css Cheat Sheet

Border

La propiedad border define el borde de un elemento, su color, su estilo y grosor.

En este div con clase border hemos definido un borde de solido de 5 pixeles color verde.

Border-left, border-right, border-top, border-bottom

Con estas propiedades de border podemos definir el borde, color, estilo y grosor de los lados (top, bottom, right, left).

En este div con clase border-sides hemos definido diferentes tipos de borde para los 4 lados del div.

Border-radius

La propiedad border-radius permite definir qué tan redondeadas serán las esquinas del div en este caso.

En este div definimos 30px de redondez a las esquinas.

Background-image

La propiedad background-image se emplea para establecer la imagen de fondo de un elemento. La imagen se muestra en la zona que ocupan el contenido y el relleno del elemento, justo hasta su borde.

En este div pusimos una imagen de fondo.

Color

La propiedad de color selecciona el valor de color de primer plano del contenido del elemento al que se aplique.

En este div definimos un color azul para el contenido.

Font

La propiedad font permite establecer algunas o todas las propiedades tipográficas del texto de forma simultánea.

En este ejemplo, aplicamos el font sans-serif al texto dentro del div.

Font-family

La propiedad font-family establece la fuente o tipo de letra con la que se muestra el texto.

Para el texto dentro del div asignamos el font-family "Gill Sans Extrabold"

font-style, font-weight

Estas propiedades nos permiten dar estilo a la letra.

Aplicamos ambas prioridades de css al contenido de este div.

Width, Height

La propiedad CSS ancho establece el ancho de un elemento. Por defecto, establece el ancho del área de contenido, pero si el tamaño del cuadro se establece en cuadro de borde, establece el ancho del área del borde. Por otro lado, la propiedad height establece la altura de un elemento.

Aplicamos un ancho de 200px y una altura de 300px a este div.

list-stlye

La propiedad de estilo de lista permite configurar el estilo de los elementos de una lista, ya sea ordenada o desordenada.

  • dos
  • tres

margin

La propiedad de margen establece el área de margen en los cuatro lados de un elemento. El margen es el espacion que separa a un elemento de otro en el flujo de un documento html.

A este div le asignamos un margen de 50px

padding

La propiedad de padding establece el espacio que existe entre el contenido de un elemento y su borde.

A este div le asignamos un padding de 50px

overflow

La propiedad CSS overflow especifica: si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque.

display

La propiedad display establece como se va a tratar un elemento (bloque, inline, flex, grid, etc)

position

La propiedad position establece cómo se coloca un elemento en un documento. Las propiedades superior, derecha, inferior e izquierda determinan la ubicación final de los elementos posicionados. Hay varios tipos de posición en css (absolute, fixed, relative, etc)

filter

La propiedad CSS filter dota de efectos gráficos como el desenfoque o cambio de color en la representación antes de que se muestre el elemento. Los filtros se utilizan comúnmente para ajustar la representación de imágenes, fondos o bordes.
